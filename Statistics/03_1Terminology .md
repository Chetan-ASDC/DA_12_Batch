## Terminology (Grouped Data)

• Class Interval  
The range of values in which data is grouped (example: 10–20).

• Frequency (f)  
The number of observations in each class interval.

• Class Midpoint (x)  
The middle value of a class interval.  
x = (Lower Limit + Upper Limit) / 2

• Lower Limit  
The smallest value of a class interval.

• Upper Limit  
The largest value of a class interval.

• Class Width (h)  
The difference between the upper limit and lower limit of a class interval.

• Cumulative Frequency (cf)  
The running total of frequencies.

• Total Frequency (Σf)  
The sum of all frequencies.

• fx  
Product of frequency and class midpoint.

• Mean (x̄)  
The average value of grouped data.  
Mean = Σfx / Σf

• Median  
The middle value of grouped data.

• Median Class  
The class interval that contains the median value.

• Mode  
The value that occurs most frequently.

• Modal Class  
The class interval with the highest frequency.

• f₀  
Frequency of the class before the modal class.

• f₁  
Frequency of the modal class.

• f₂  
Frequency of the class after the modal class.

• Range  
The difference between the highest and lowest values.

• Variance  
A measure of how far values are from the mean.

• Standard Deviation  
The square root of variance, showing data spread.
