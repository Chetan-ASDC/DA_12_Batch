# Descriptive Statistics (Grouped Data) 

## What is Grouped Data?
When a large amount of data is arranged into **class intervals** with their frequencies, it is called **grouped data**.  
Exact values are not known, so we use **midpoints** for calculations.

---

## Example of Grouped Data

| Class Interval | Frequency (f) |
|---------------|---------------|
| 10 – 20 | 5 |
| 20 – 30 | 9 |
| 30 – 40 | 14 |
| 40 – 50 | 8 |
| **Total (Σf)** | **36** |

---

## Step 1: Class Midpoint (x)

**Formula:**
```
x =
Lower Limit + Upper Limit
-------------------------
            2
```

| Class Interval | f | Midpoint (x) |
|---------------|---|--------------|
| 10 – 20 | 5 | 15 |
| 20 – 30 | 9 | 25 |
| 30 – 40 | 14 | 35 |
| 40 – 50 | 8 | 45 |

---

## 1. Mean (Average)

### Meaning
Mean is the **average value** of the data.

### Formula
```
Mean =
      ∑fx
     -----
      ∑f

```
### Calculation

| f | x | fx |
|---|---|----|
| 5 | 15 | 75 |
| 9 | 25 | 225 |
| 14 | 35 | 490 |
| 8 | 45 | 360 |
| **Σf = 36** | | **Σfx = 1150** |

```
Mean = 1150 / 36 = 31.94
```

---

## 2. Median

### Meaning
Median is the **middle value** of the data.

### Formula
```

Median = l + ((N/2 − cf) / f) × h

```

### Values
- \(N = 36 \Rightarrow N/2 = 18\)
- Median class = **30 – 40**
- \(l = 30\)
- \(c_f = 14\)
- \(f = 14\)
- \(h = 10\)

### Calculation
```
Median = 30 + ((18 − 14) / 14) × 10

Median = 32.86

```

---

## 3. Mode

### Meaning
Mode is the value that occurs **most frequently**.

### Formula
```
Mode = l + ((f1 − f0) / (2f1 − f0 − f2)) × h
```

### Values
- Modal class = **30 – 40**
- \(l = 30\)
- \(f_1 = 14\)
- \(f_0 = 9\)
- \(f_2 = 8\)
- \(h = 10\)

### Calculation
```

Mode = 30 + ((14 − 9) / (28 − 9 − 8)) × 10

Mode = 35.56

```

---

## 4. Range

### Meaning
Range shows how **spread out** the data is.

### Formula
```
Range=Highest Value−Lowest Value

```
### Calculation
```
Range=50−10=40

```
---

## 5. Variance

### Meaning
Variance measures how far values are from the mean.

### Formula
```
Variance =
           ∑f(x − x̄)²
          -------------
               ∑f

```

**Steps:**
1. Find mean  
2. Subtract mean from midpoint  
3. Square the result  
4. Multiply by frequency  
5. Divide by total frequency  

---

## 6. Standard Deviation

### Meaning
Standard deviation shows the **average spread** of data from the mean.

### Formula
```

Standard Deviation = √Variance

```

- Small SD → data close to mean  
- Large SD → data widely spread  

---

## Quick Summary

| Measure | Meaning |
|-------|--------|
| Mean | Average |
| Median | Middle value |
| Mode | Most frequent value |
| Range | Max − Min |
| Variance | Spread from mean |
| Standard Deviation | Square root of variance |
